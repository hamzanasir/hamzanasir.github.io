Diborane Manifold

The Diborane Manifold is a manfiold constructed for the purpose of handling and reacting toxic gasses. It consists of Flow Controllers, Pressure Transducers, Thermocouples and Actuated/Manual Valves all connected to a PLC. The interface on the right communciates with these devices in order to remotely control the behaviour of the system. The operator can:

  • Open valves by clicking on them
  • Set flow rate of flow controllers
  • See which gasses occupy which lines by the multi-colored pipes using a BFS algorithm
  • Observe and respond to any irregularities in the system
